# 💼 Afjal Hayat — Personal Portfolio Website

Welcome to my **personal portfolio website** — a fully responsive, interactive, and fast React-based web application that showcases my projects, skills, and contact details.

🔗 **Live Demo**: [https://afjalhayat.netlify.app](https://afjalhayat.netlify.app)

---

## 🚀 Tech Stack

- **Frontend Framework**: [React.js](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **State Management**: React Hooks (`useState`, `useRef`)
- **Email Handling**: [EmailJS API](https://www.emailjs.com/) (Client-side HTTP API)
- **Animations & Interactions**: Toast Notifications (`react-toastify`)

---

## 📬 Features

- 📱 **Fully Responsive** across all screen sizes (Mobile, Tablet, Desktop)
- 🎨 **Modern UI/UX** with Tailwind CSS styling
- 📩 **Contact Form** with working email submission via EmailJS API
- 🧠 **State Management** using React Hooks
- ☁️ **Deployed on Netlify** for blazing-fast performance
